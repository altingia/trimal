50  /**
51  * \brief Class to calculate the consistency between several MSA containing the same sequences, differently aligned.\n
52  * Using this statistics, the class is able to select the most consistent alignment between all alignments provided.\n
53  * It is possible to forcefully select an alignment, but to calculate the statistics for latter use.\n
54  * After selecting an alignment (most consistent or manually selected),
55  * it is possible to use this statistic to trim the alignment, removing
56  * columns that are not consistent enough with the other alignments.
57  */

63  * \brief Method to compare a set of MSA,
64  * all containing the same sequences and residues.\n
65  * The number of residues must be the same, but gaps are not taken into account.\n
66  * This is due to the same sequence being aligned in different ways,
67  * which changes the gap patterns.
68  * \param comparesetFilePath Path to the file containing paths for each alignment to compare.\n One per line
69  * \param formatManager Format manager, to load and save the alignments.
70  * \param manager trimAl manager, to store the choosen alignment in trimAlManager::origAlig
71  * \param forceFile path to file to forcefully select. If nullptr, the most consistent alignment will be selected.
72  *
73  * \note The method does not return anything, as the alignment is stored on trimAlManager::origAlig
